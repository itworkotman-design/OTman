# Legal Content

## Source

- `lib/content/LegalContent.ts`

## Responsibility

Provides localized privacy policy and terms and conditions copy for the public legal pages.

## Functions

| Function | Description |
| --- | --- |
| `privacyPolicyContent` | Stores the localized privacy policy title, intro, update date, sections, and list items. |
| `termsContent` | Stores the localized terms and conditions title, intro, update date, sections, and list items. |
