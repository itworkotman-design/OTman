# Tjenester Route

## Source

- `app/(site)/[locale]/tjenester/page.tsx`

## Responsibility

Defines the localized public `/tjenester` route and renders the Tjenester page with localized content.

## Functions

| Function | Description |
| --- | --- |
| `Page` | Loads the active locale from route params and renders `Tjenester` with `TjenesterContent`. |
