# Tjenester

## Source

- `app/_components/site/pageComponents/Tjenester.tsx`

## Responsibility

Renders the public Tjenester page form and content sections. The page uses `TjenesterContent` for localized copy and still submits inquiries to the existing public manpower inquiry endpoint.

## Functions

| Function | Description |
| --- | --- |
| `Tjenester` | Renders the localized page, validates the request form, submits the inquiry, and displays submission status. |
