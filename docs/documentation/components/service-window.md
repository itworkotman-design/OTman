# Service Window

## Source

- `app/_components/site/TransportService/ServiceWindow.tsx`

## Responsibility

Renders the home page service carousel/grid and routes clickable service cards to public service pages. The manpower service card now links to `/tjenester`.

| Function | Description |
| --- | --- |
