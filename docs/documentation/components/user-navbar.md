# User Navbar

## Source

- `app/_components/Users/Navbar.tsx`

## Responsibility

Renders the non-admin booking sidebar with profile details, S3-backed logo display support, booking links, language switch, feature request trigger, and logout action.

## Functions

| Function | Description |
| --- | --- |
| `Icon` | Renders the sidebar navigation SVG icons. |
| `UserNavbar` | Renders the booking navigation for order creators and subcontractors, displays the current user's managed logo through the profile appearance helper, applies the current language to labels, and exposes the language switch above the request-function action. |
