# Root Layout

## Source

- `app/layout.tsx`

## Responsibility

Loads global styles, defines fallback application metadata, and renders the root HTML and body shell for the app.

## Functions

| Function | Description |
| --- | --- |
| `metadata` | Defines fallback metadata using the Otman AS name for routes that do not provide their own metadata. |
| `RootLayout` | Renders the root HTML and body wrapper around all app routes. |
