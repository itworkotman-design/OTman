# Footer

## Source

- `app/_components/site/Footer.tsx`

## Responsibility

Renders the public site footer with company information, localized navigation links, localized legal links, and copyright text.

## Functions

| Function | Description |
| --- | --- |
| `Footer` | Renders the footer and prefixes internal links with the active locale. |
| `localizeHref` | Converts internal footer links into locale-aware URLs while leaving external URLs unchanged. |
